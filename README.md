## 코드 명명법

| 이름 | 설명 |
| --- | --- |
| `short.*` | 숏코딩 풀이 |
| `solution.*` | 최대한 이해하기 쉽게 쓴 풀이 |
| `submit.*` | 프로그래머스 사이트에 등록되어 있는 풀이 |
